  • 表面検査 / Surface Inspection
    環境Borland C++ Builder
    Microsoft Visual Studio
    言語C++
    C#
    技術Spring.NET
    db4o

    長瀬産業（株）社製のSCANTECという 表面検査装置と連携するWindowsアプリケーションを開発しています。 表面検査装置には、検体を撮影するカメラが接続されています。

    カメラから入力した映像を表面検査装置内で様々な画像処理を施し、欠陥を検出します。

    • 様々な品種に対応するために、品種ごとの条件を設定
    • 表面検査装置で検出した欠陥を、アプリケーションで受信／記録
    • 欠陥をマップや数値で表示したり、欠陥種別ごとに分類し品質の判定
    • ディジタルI/Oやエンコーダ、PLCなどの外部機器の制御
    • 品質管理のための集計やレポートを出力
    • 検体上の実際の欠陥へマーキングやラベルなどの出力
